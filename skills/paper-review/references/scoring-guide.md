# 评分标准与接收建议指南

## 维度评分标准（1-5 分）

### 通用评分定义
- **5/5 优秀**：质量突出，无显著问题
- **4/5 良好**：有小问题但不影响核心贡献
- **3/5 可接受**：有明显不足但可通过修改解决
- **2/5 低于标准**：存在重大问题需要大幅修改
- **1/5 不可接受**：存在根本性缺陷

### 各维度评分细则

#### 1. 摘要与动机
| 分数 | 标准 |
|------|------|
| 5 | 研究空白定义精确，有充分的量化证据支撑 |
| 4 | 动机清晰，有文献支撑 |
| 3 | 动机存在但支撑不够充分 |
| 2 | 研究动机模糊或与现有工作区别不明 |
| 1 | 无法理解为什么需要这项工作 |

#### 2. 新颖性
| 分数 | 标准 |
|------|------|
| 5 | 提出全新范式或框架，开创新方向 |
| 4 | 有实质性创新，非简单技术组合 |
| 3 | 有一定创新但主要是已知技术的组合 |
| 2 | 增量式改进，创新性有限 |
| 1 | 与已发表工作高度相似，无明显创新 |

#### 3. 技术正确性
| 分数 | 标准 |
|------|------|
| 5 | 所有公式和推导完全正确，假设明确合理 |
| 4 | 技术上基本正确，个别小错误不影响结论 |
| 3 | 存在技术问题但核心方法可能成立 |
| 2 | 重要公式有错误或关键假设不合理 |
| 1 | 核心方法存在根本性技术缺陷 |

#### 4. 实验
| 分数 | 标准 |
|------|------|
| 5 | 全面的实验：最新基线、多数据集、消融、统计显著性、效率分析 |
| 4 | 实验充分，基线较新，有消融实验 |
| 3 | 实验基本够用但缺少部分重要对比 |
| 2 | 基线过时或对比不公平，缺少消融 |
| 1 | 实验不足以支撑结论 |

#### 5. 写作质量
| 分数 | 标准 |
|------|------|
| 5 | 语法完美，结构清晰，符号一致，图表美观 |
| 4 | 写作流畅，偶有小错 |
| 3 | 可读但有明显语法问题或结构混乱 |
| 2 | 写作质量差，影响理解 |
| 1 | 语言障碍严重，难以理解论文内容 |

#### 6. 可复现性
| 分数 | 标准 |
|------|------|
| 5 | 提供代码 + 详细超参数 + 训练细节，可直接复现 |
| 4 | 实现细节充分，有经验的研究者可复现 |
| 3 | 部分细节缺失但核心方法可理解 |
| 2 | 关键实现细节缺失 |
| 1 | 无法根据论文内容复现 |

#### 7. 相关工作
| 分数 | 标准 |
|------|------|
| 5 | 全面覆盖近年关键工作，讨论充分，定位清晰 |
| 4 | 覆盖主要工作，有适当讨论 |
| 3 | 覆盖不够全面，遗漏部分重要工作 |
| 2 | 遗漏多篇高引论文，讨论不足 |
| 1 | 相关工作严重不足或完全缺失 |

#### 8. 补充材料
| 分数 | 标准 |
|------|------|
| 5 | 详尽的附录：额外实验、证明、可视化 |
| 4 | 有有价值的补充信息 |
| 3 | 补充材料存在但信息量有限 |
| 2 | 补充材料质量差或不相关 |
| 1 | 无补充材料且论文正文不足 |

## 接收建议指南

| 建议 | 条件 |
|------|------|
| **Strong Accept** | 前 5% 质量，重大贡献，无主要问题 |
| **Accept** | 明确贡献，执行扎实，仅有小问题 |
| **Weak Accept** | 工作有趣但有明显不足，边缘质量 |
| **Borderline** | 优缺点相当，可接受可不接受 |
| **Weak Reject** | 有一定价值但问题大于贡献 |
| **Reject** | 技术或实验有重大缺陷，贡献不足 |
| **Strong Reject** | 多个维度存在根本性问题 |

## 审稿置信度

| 级别 | 含义 |
|------|------|
| **5/5 专家** | 在该子领域有发表论文经验 |
| **4/5 自信** | 熟悉该领域，已验证关键声明 |
| **3/5 较自信** | 具备一般知识，部分声明未验证 |
| **2/5 不确定** | 非主要专业领域 |
| **1/5 低置信** | 不熟悉该领域 |
